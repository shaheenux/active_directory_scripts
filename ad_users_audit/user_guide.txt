=====================================================
USER GUIDE: WINDOWS LOCAL AUDITOR
=====================================================

1. OPERATION MODE
-----------------------------------------------------
The script is set to DEFAULT mode. It does NOT:
- Change passwords.
- Delete users.
- Modify permissions.

It ONLY reads and reports data.

2. RUNNING ON KALI (CROSS-PLATFORM)
-----------------------------------------------------
If you are on Kali Linux and want to run this against
a Windows machine:
- Ensure the Windows machine has WinRM enabled.
- Use 'pwsh' (PowerShell for Linux) to execute.

3. DATA INTERPRETATION
-----------------------------------------------------
- Password_Age_Days: If this value is >90, consider 
  enforcing a password change.
- Enabled=False: These are dormant accounts. Ensure 
  they are not "backdoor" accounts left by old software.

4. LOGS
-----------------------------------------------------
All reports are saved in the 'Audit_Reports' folder.
These can be opened in Excel, LibreOffice, or any
text editor.
